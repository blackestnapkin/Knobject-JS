Welcome to the Knobject.js tutorial.

First, we have to create our knob canvas:

<body>
<canvas id="myKnob" width="30" height="30" onmousedown="knob.capture(event)"></canvas>
</body>

Rememeber, that we need to define the width and height of the canvas to properly work with it.
Later, we will pass the width into the Knob object's diameter property.
Also note, the inline onmousedown function. Let's break it down:

"knob" is going to refer to our variable that we have save our knob object to.
".capture()" is the function the knob will run internally.
"event" is needed to determine which direction the knob rotates. Moving the
mouse up, turns it right and down, turns it left.

This is our resulting canvas. As you can see, it is currently blank.

Next, we have to create the instance of our knob in Javascript.
We also have to give our knob some input to connect it to our canvas.
And finally, we need a pair of custom functions for our knob to perform.

<script>
var knob = new Knob();
var left = function(value) { document.getElementById('value').innerHTML = value; }
var right = function(value) { document.getElementById('value').innerHTML = value; }
var array = { diameter: 30, location: 'myKnob', leftFunction: left, rightFunction: right }
knob.init(array);
</script>
