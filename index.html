Welcome to the Knobject.js tutorial.

Parameters:

defaultValue: the the default position of the knobrelative to your min and max

diameter: the diameter of the knob in pixels; the default is '50px'

knobContainer: the id of the div housing the knob

knobId: the id of the canvas generated for the knob

max: the maximum value that 'value' can have

min: the minimum value that 'value' can have

onCenter: the function called when the knob is reset(by alt/option clicking), knobValue is passed as argument

  NOTE: onCenter will only be called if knob orientaion is 'centered'

onLeftTurn: the function called when the knob is turned left, knobValue is passed as argument

onRightTurn: the function called when the knob is turned right, knobValue is passed as argument

orientation: defines whether the knob's default position is all the way down ('normal') or upright ('centered')

value: the position of the knob
