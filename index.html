Welcome to the Knobject.js tutorial.

First, we have to create our knob canvas:

<body>
<canvas id="myKnob" width="30" height="30"></canvas>
</body>

Rememeber, that we need to define the width and height of the canvas to properly work with it.
Later, we will pass the defined width into the Knob object's diameter property.

This is our resulting canvas. As you can see, it is currently blank.

Next, we have to create the instance of our knob in Javascript:

<script>
var knob = createKnob();
</script>

Now, we have to give our knob some input to connect it to our canvas.
The bare minimum we need is the diameter and the canvas id.

<script>
var knob = createKnob();
var array = { diameter: 30, location: 'myKnob' }
knob.init(array);
</script>
