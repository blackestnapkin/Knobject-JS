Next, we have to initialize our knob in Javascript.


diameter: the diameter of the knob in pixels

knobContainer: the id of the div housing the knob

knobId: the id of the canvas generated for the knob

knobValue: the position of the knob

max: the maximum value that knobValue can have

min: the minimum value that knobValue can have

onLeftTurn: the function called when the knob is turned left, knobValue is passed in as an argument

onRightTurn: the function called when the knob is turned right, knobValue is passed in as an argument

type: defines either 'smooth' or 'stepped' taper
